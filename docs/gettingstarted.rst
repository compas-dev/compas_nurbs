Installation
============

On Mac
------

.. code-block:: bash

    conda create -n nurbs python=3.7 python.app COMPAS">=0.16.3"
    conda activate nurbs
    pip install -e .

On Windows
----------

.. code-block:: bash

    conda create -n nurbs python=3.7 COMPAS">=0.16.3"
    conda activate nurbs
    pip install -e .
